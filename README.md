# Bodhi-DeepResearch

Bodhi-DeepResearch is our research-focused system designed to perform deep analytical reasoning, structured synthesis, and high-quality long-form generation across complex research tasks.

This repository contains implementation details, evaluation artifacts, and benchmark results for the Bodhi-DeepResearch system.

---

## 📊 DeepResearch Bench Evaluation Results

We evaluated **Bodhi-DeepResearch** on the DeepResearch Bench to assess performance across multiple qualitative dimensions of research capability.

### Results Summary

| Dimension               | Score  |
|-------------------------|--------|
| Comprehensiveness       | 54.41  |
| Insight                 | 56.36  |
| Instruction Following   | 53.11  |
| Readability             | 52.10  |
| **Overall Score**       | **54.45** |

### Evaluation Dimensions Explained

- **Comprehensiveness** – Measures coverage depth and completeness of research responses.
- **Insight** – Evaluates originality, reasoning quality, and depth of analysis.
- **Instruction Following** – Assesses alignment with task constraints and prompt requirements.
- **Readability** – Measures clarity, structure, and communication quality.
